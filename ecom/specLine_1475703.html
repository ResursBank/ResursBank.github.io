Resurs Bank e-Commerce Platform : specLine

Created by Benny, last modified by Tobias on 2019-06-24

The payment line (item) specification. These can be used to provide detailed information about the contents of the payment.
Contains elements as defined in the following table.

ComponentTypeOccursNillable?Description
idid1..1NoThe line identity
artNostring1..1NoArticle number of the item. (Max 100 characters)
descriptionstring1..1YesThe item description.
quantitydecimal1..1NoThe line quantity.
unitMeasurestring1..1No


unitAmountWithoutVatdecimal1..1NoThe unit amount without VAT.
vatPctpercent1..1NoThe VAT percentage.
totalVatAmountdecimal1..1NoThe total item VAT amount.
totalAmountdecimal1..1NoThe total item amount, including VAT.
deliveryDatedate0..1NoEstimated delivery date
