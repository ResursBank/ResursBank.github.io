# pdf 

A PDF document  
Contains elements as defined in the following table.

| Component | Type         | Occurs | Nillable? | Description               |
|-----------|--------------|--------|-----------|---------------------------|
| name      | string       | 1..1   | No        | The name of the document. |
| pdfData   | base64Binary | 1..1   | No        | The document data.        |
