# paymentStatus 
Created by Benny, last modified on 2013-06-18
  
| Status Code | Description      |
|-------------|------------------|
| DEBITABLE   | Can be debited.  |
| CREDITABLE  | Can be credited. |
| IS_DEBITED  | Is debited.      |
| IS_CREDITED | Is credited.     |
| IS_ANNULLED | Is annulled      |
